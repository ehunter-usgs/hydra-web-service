Event Terms

depth
Data Type
Decimal
Typical Values
[0, 1000]
Description
Depth in kilometers.
latitude
Data Type
Decimal
Typical Values
[-90.0, 90.0]
Description
Decimal degrees latitude. Negative values for southern latitudes.
longitude
Data Type
Decimal
Typical Values
[-180.0, 180.0]
Description
Decimal degrees longitude. Negative values for western longitudes.
time
Data Type
String
Typical Values
[-180.0, 180.0]
Description
ISO8601 formatted event time.

Magnitude Terms

associated-by
Data Type
String
Typical Values
"wphase_module", "mb_module"
Description
The name of the author who associated this magnitude with this event.
associated-by-installation
Data Type
String
Typical Values
"NEIC", "HON"
Description
The code for the installation of the author who associated this magnitude with this event.
author
Data Type
String
Typical Values
"wphase_module", "mb_module"
Description
The name of the author who created this magnitude.
derived-magnitude
Data Type
Decimal
Typical Values
5.547000000000001, 5.744000000000001
Description
The computed magnitude value.
derived-magnitude-type
Data Type
String
Typical Values
"Mww", "mb"
Description
The computed magnitude type.
id
Data Type
String
Typical Values
"50003UWR/wphase_module/NEIC/Mww", "50003UWR/mb_module/NEIC/mb"
Description
The logical id for this magnitude.
installation
Data Type
String
Typical Values
"NEIC", "HON"
Description
The code for the installation of the author created this magnitude.
is-internal
Data Type
Boolean
Typical Values
true, false
Description
A flag indicating whether this magnitude was produced by an internal author.
is-preferred-for-type
Data Type
Boolean
Typical Values
true, false
Description
A flag indicating whether this magnitude is preferred for its type.
num-stations-associated
Data Type
Integer
Typical Values
[0, 1000]
Description
The number of stations associated with this magnitude.
num-stations-used
Data Type
Integer
Typical Values
[0, 1000]
Description
The number of stations that were used in generating the derived magnitude.
