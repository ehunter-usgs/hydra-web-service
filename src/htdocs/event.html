Event Endpoint Documentation

Summary information for one event.

Parameters

All parameters are required

huid
Hydra unique identifier.
Example: 50003UWR

Response

Responses use JSON format.

Response format documentation

Event Feature Properties

eventtime
ISO8601 formatted string.

Time of event

magnitude
Number.

Preferred magnitude value.

magnitudeType
String.

Type of magnitude calculation used to determine magnitude

magnitudes
Array<Object>

Array of magnitudes associated to event. Each magnitude object has the following properties:

author
String.

Author of magnitude

id
String.

Unique identifier for magnitude.

installation
String.

Installation of author, essentially a namespace for authors.

type
String.

Magnitude Type

url
String.

site-root relative URL to request details from magnitude endpoint.

value
Number.

Magnitude value

title
String.

Title of event

type
String.

Event type, most commonly earthquake

Example

Example Request URL

event.json?huid=50003UWR
